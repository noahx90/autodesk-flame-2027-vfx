## How to Use It

Autodesk Flame is commonly placed inside a post-production pipeline where editorial, compositing, and finishing tasks overlap.

Typical workflow examples:
- Import source media and organize it with metadata
- Build node-based composites for shot-level adjustments
- Perform keying, rotoscoping, and cleanup on selected elements
- Apply HDR color grading to match shots across a sequence
- Use motion graphics and 3D compositing for layered finishing
- Run batch operations when multiple assets need the same treatment
- Collaborate with other team members during review and delivery

For best results, choose the workflow order that matches your production needs, whether that starts with editorial assembly, shot cleanup, or the final grade.
